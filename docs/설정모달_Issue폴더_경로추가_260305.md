# 설정 모달 Issue 폴더 경로 추가 (2026-03-05)

## 변경 배경
- 설정 화면에서 Vault 경로만 변경 가능하고, Issue 폴더는 코드/환경변수에 의존했습니다.
- UI에서 Issue 저장 위치를 즉시 바꿀 수 없어 운영 유연성이 낮았습니다.

## 적용 내용
- `web/static/index.html`
  - 설정 모달에 `settings-issue-folder-input` 입력 필드 추가
- `web/static/app.js`
  - `loadConfig()`에서 `issue_folder` 값을 입력 필드에 채움
  - `saveSettings()`에서 `vault_path + issue_folder`를 함께 전송
- `web/config.py`
  - `set_issue_folder()` 추가 (설정 파일 저장)
- `web/routers/files.py`
  - `/api/config` 요청 바디에 `issue_folder` 지원
  - 상대 경로 검증 추가
    - 절대경로 금지
    - `..` 상위 경로 금지
    - 빈 값 금지

## 기대 효과
- 서버 재시작 없이 UI에서 Issue 저장 경로를 변경할 수 있습니다.
- 잘못된 경로 입력을 서버에서 차단해 안전하게 설정을 유지할 수 있습니다.
