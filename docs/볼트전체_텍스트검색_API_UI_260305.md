# 볼트 전체 텍스트 검색 API + UI (2026-03-05)

## 변경 배경
- 기존 검색은 파일명 필터만 지원하여 파일 내용 기반 탐색이 어려웠습니다.
- 키워드가 포함된 문서를 빠르게 찾기 위해 본문 검색 기능이 필요했습니다.

## 적용 내용
- `web/routers/files.py`
  - `GET /api/search` 엔드포인트 추가
    - 파라미터: `q`, `limit`
    - 확장자 대상: `.md`, `.txt`
    - 결과: `path`, `name`, `snippet`
  - 검색어 미입력/잘못된 limit에 대한 검증 추가
- `web/static/index.html`
  - 헤더에 `검색` 버튼 추가
  - 볼트 내용 검색 모달(`content-search-modal`) 추가
- `web/static/app.js`
  - `openContentSearchModal()`
  - `runContentSearch()`
  - `renderContentSearchResults()`
  - 결과 클릭 시 해당 파일 열기(`openFile`)
  - 단축키 `Ctrl/Cmd + Shift + F` 추가
- `web/static/style.css`
  - 검색 모달 입력/결과 리스트 스타일 추가

## 기대 효과
- 파일 내용 키워드로 볼트 전체를 빠르게 탐색할 수 있습니다.
- 검색 결과 스니펫을 보고 바로 문서로 이동하는 워크플로우가 가능해집니다.
