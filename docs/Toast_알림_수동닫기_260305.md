# Toast 알림 수동 닫기 개발 내역

> 개발일: 2026-03-05

---

## 개요

Toast 알림에 수동 닫기 기능을 추가하고, 에러 알림의 자동 닫기를 비활성화했습니다.

## 변경 사항

| 파일 | 변경 내용 |
|------|----------|
| `web/static/app.js` | `toast()` 함수에 닫기 버튼 추가, 에러 Toast 자동 닫기 비활성화, `role="alert"` 추가 |
| `web/static/style.css` | `.toast-close`, `.toast-message` 스타일 추가, Toast 레이아웃 flex로 변경 |

## 주요 기능

1. **수동 닫기**: X 버튼 또는 Toast 클릭으로 즉시 닫기
2. **에러 알림 유지**: `type="error"` Toast는 자동 닫기 비활성화 (사용자가 직접 닫아야 함)
3. **일반 알림 자동 닫기**: 성공/정보 알림은 기존과 같이 3.5초 후 자동 닫기
4. **접근성**: `role="alert"` 및 `aria-label` 추가
