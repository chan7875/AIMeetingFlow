## 2. UI/UX 개선 권고사항

### 2.1 검색 기능 부재 (우선도: 높음)

**현황**: 사이드바 트리에서 파일을 찾으려면 수동으로 폴더를 열어야 함.

**권고**:
- 사이드바 상단에 파일명 검색/필터 입력 추가
- 입력 시 트리를 실시간 필터링 (fuzzy matching)
- `Ctrl+K` 또는 `Cmd+K` 단축키로 검색 포커스

### 2.2 접근성(Accessibility) 개선 (우선도: 높음)

**현황**: ARIA 레이블 부재, 키보드 네비게이션 미지원, 포커스 아웃라인 없음.

**권고**:
- 트리 아이템에 `role="treeitem"`, 컨테이너에 `role="tree"` 추가
- 모달에 `role="dialog"`, `aria-modal="true"` 추가
- 접기/펼치기 요소에 `aria-expanded` 속성 추가
- `:focus-visible` 스타일 추가로 키보드 사용자 시각 피드백 제공
- 트리에서 방향키(↑↓) 네비게이션 지원

### 2.3 Toast 알림 개선 (우선도: 중간)

**현황**: 3.5초 고정 타임아웃, 수동 닫기 불가, 긴 메시지 잘림.

**권고**:
- 클릭 또는 X 버튼으로 수동 닫기 추가
- 에러 Toast는 자동 닫기 비활성화 (사용자가 직접 닫도록)
- 긴 메시지는 말줄임 + "자세히" 링크

### 2.4 파일 관리 기능 (우선도: 중간)

**현황**: 파일 조회/업로드/다운로드만 가능. 삭제/이름변경/이동 불가.

**권고**:
- 트리 아이템 우클릭 컨텍스트 메뉴: 다운로드, 이름변경, 삭제
- 삭제 시 확인 다이얼로그 필수
- 드래그 앤 드롭으로 파일 이동

### 2.5 마크다운 뷰어 강화 (우선도: 중간)

**현황**: 기본 마크다운 렌더링 + 코드 하이라이팅만 지원.

**권고**:
- 헤더(H1~H3) 기반 자동 목차(TOC) 생성
- 헤더 클릭 시 앵커 링크로 스크롤 이동
- Mermaid 다이어그램 렌더링 지원 검토
- 이미지 클릭 시 확대(lightbox) 기능

### 2.6 모바일 경험 개선 (우선도: 중간)

**현황**: 기본 반응형은 지원하나 세부 조정 부족.

**권고**:
- 헤더 버튼 폰트 12px → 14px (터치 타겟 확대)
- AI 패널 기본 높이 320px → 모바일에서 40vh로 조정
- 컨텐츠 하단에 safe-area padding 추가 (노치 대응)
- 사이드바 드로어 전환 시 하드웨어 가속(`transform: translate3d`) 활용

### 2.7 색상 대비 및 가독성 (우선도: 낮음)

**현황**: `--text-dim: #7f849c`가 배경(`--surface: #252536`) 대비 WCAG AA 기준 미달 가능성.

**권고**:
- `--text-dim`을 `#9399b2` 이상으로 밝게 조정
- 또는 dim 텍스트 사용 위치를 `--surface2` 배경과 조합하여 대비 확보

### 2.8 라이트 테마 지원 (우선도: 낮음)

**현황**: 다크 테마만 존재.

**권고**:
- CSS Variables 기반으로 라이트 테마 세트 정의
- 헤더에 테마 토글 버튼 추가
- `prefers-color-scheme` 미디어 쿼리로 시스템 설정 연동

## 3. 금일 수정 완료 항목 (2026-03-04)

### 3.1 파일 다운로드 기능 추가

| 파일 | 변경 내용 |
|------|----------|
| `web/routers/files.py` | `GET /api/download?path=` 엔드포인트 추가 (`FileResponse`, `Content-Disposition: attachment`) |

| `web/static/app.js` | `downloadFile()`, `downloadCurrentFile()` 함수 추가 |

| 파일 | 변경 내용 |
|------|----------|
| `web/routers/files.py` | `_build_tree()`에서 `.pptx` 확장자 포함 |
| `web/static/app.js` | `.pptx` 파일에 📊 아이콘 + ⬇ 표시, 클릭 시 바로 다운로드 |

### 3.3 자동 감시(Auto-Watch) 슬라이드 루프 버그 수정

**문제**: `Slides` 폴더의 `_source_*.md` 임시 파일을 auto-watch가 감지 → 재처리 → 슬라이드 재생성 무한 루프.

| 파일 | 변경 내용 |
|------|----------|
| `web/routers/ai.py` | `AUTO_WATCH_EXCLUDED_DIR_NAMES`에 `"Slides"` 추가 |
| `web/routers/ai.py` | `_scan_watch_candidates()`에서 `_source_` 접두어 파일 필터링 |
