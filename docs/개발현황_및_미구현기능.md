## 1. 구현 완료 기능

### 웹 뷰어 (web/)

| 기능 | 파일 | 설명 |
|------|------|------|
| 파일 트리 탐색 | `routers/files.py` | 볼트 디렉토리 트리 조회, 마크다운/텍스트 필터링 |
| 마크다운 렌더링 | `static/app.js` | marked.js + highlight.js 기반 렌더링 |
| AI 분석 (스트리밍) | `routers/ai.py` | Claude / Codex CLI SSE 실시간 스트리밍 |
| Issue 자동 저장 | `routers/ai.py` | 번호 채번, 템플릿 적용, account별 폴더 저장 |
| Auto-watch | `routers/ai.py` | 새 파일 감지 → Codex 자동 처리 (2초 폴링) |
| Git 연동 | `routers/git.py` | pull / push / commit 웹 UI에서 실행 |
| 파일 업로드 | `routers/files.py` | 최대 50MB, 허용 확장자 화이트리스트 |
| 설정 관리 | `web/config.py` | 볼트 경로, issue 폴더, auto-watch 상태 저장 |

### AI 서비스 (services/)

| 서비스 | 파일 | 설명 |
|--------|------|------|
| ChatGPT 콘텐츠 생성 | `chatgpt_service.py` | YouTube SRT → 챕터/썸네일/쇼츠/태그 JSON 생성 |
| ChatGPT 뉴스 생성 | `chatgpt_service.py` | 텍스트 → 유튜브 메타데이터 JSON 생성 |
| ChatGPT Threads | `chatgpt_service.py` | 450자 Threads 포스트 생성 |
| Claude CLI | `claude_cli_service.py` | 세션 관리, 재사용, 스트리밍 지원 |
| Codex CLI | `codex_cli_service.py` | 세션 없이 단순 실행, 스트리밍 지원 |

## 2. 미구현 기능

### 2-1. 의존성 누락 (즉시 수정 필요)

| 항목 | 영향 | 내용 |
|------|------|------|
| `config/settings.py` | `services/` 전체 | `from config.settings import settings` import 실패 |
| `models/prompt_settings.py` | `chatgpt_service.py` | `get_prompt_profile()` 함수 없음 |

### 2-2. 플랫폼 배포 서비스 (services/ 미구현)

| 서비스 | 우선순위 | 내용 |
|--------|----------|------|
| `naver_service.py` | 높음 | Playwright 기반 네이버 블로그 자동 포스팅 (Chrome CDP) |
| `youtube_service.py` | 높음 | OAuth2 업로드, yt-dlp 다운로드, ffmpeg 프레임 추출 |
| `threads_service.py` | 중간 | Meta Graph API 게시, 450자 문단 단위 자동 분할 |
| `linkedin_service.py` | 중간 | OAuth2 게시, 1~72시간 쿨다운 관리 |
| `scheduler_service.py` | 높음 | APScheduler 기반 전체 워크플로 오케스트레이션 |

### 2-3. 웹 뷰어 UI 기능 (web/)

| 기능 | 우선순위 | 내용 |
|------|----------|------|
| 파일 편집 | 높음 | 현재 읽기 전용 — 인라인 마크다운 에디터 필요 |
| 파일 삭제 / 이름 변경 | 중간 | 파일 관리 API 및 UI 미구현 |
| 전문 검색 | 중간 | 볼트 내 텍스트 검색 (파일명 + 내용) |
| AI 처리 기록 | 낮음 | 이전 AI 결과 히스토리 패널 |
| 접근 인증 | 낮음 | 로그인 없이 누구나 접근 가능 |

## 3. 권장 구현 순서

```
1단계 (기반 작업)
  └─ config/settings.py 생성 (환경변수 기반 설정 모듈)
  └─ models/prompt_settings.py 생성 (프롬프트 프로파일 관리)

2단계 (핵심 서비스)
  └─ scheduler_service.py (APScheduler 워크플로 오케스트레이션)
  └─ youtube_service.py (OAuth2 + yt-dlp + ffmpeg)
  └─ naver_service.py (Playwright + Chrome CDP)

3단계 (추가 플랫폼)
  └─ threads_service.py (Meta Graph API)
  └─ linkedin_service.py (OAuth2 + 쿨다운)

4단계 (웹 뷰어 개선)
  └─ 파일 편집 기능
  └─ 전문 검색
  └─ AI 처리 기록
```
