# 라이트 테마 지원 개발 내역

> 개발일: 2026-03-05

---

## 개요

CSS Variables 기반의 라이트 테마를 추가하고, 테마 토글 버튼 및 시스템 설정 자동 연동을 구현했습니다. 추가로 다크 테마의 `--text-dim` 색상 대비도 개선했습니다.

## 변경 사항

| 파일 | 변경 내용 |
|------|----------|
| `web/static/style.css` | `[data-theme="light"]` CSS Variables 세트 정의, `prefers-color-scheme` 미디어 쿼리, `--text-dim` 대비 개선 (#7f849c → #9399b2) |
| `web/static/index.html` | 테마 토글 버튼 추가, highlight.js 라이트/다크 CSS 이중 링크 |
| `web/static/app.js` | `initTheme()`, `toggleTheme()`, `applyTheme()` 함수 추가, localStorage 저장 |

## 주요 기능

1. **라이트 테마**: Catppuccin Latte 기반 밝은 색상 테마
2. **테마 토글**: 헤더의 테마 버튼으로 다크/라이트 전환
3. **시스템 연동**: `prefers-color-scheme: light` 미디어 쿼리로 OS 설정 자동 감지
4. **설정 저장**: localStorage에 테마 선택 저장 (새로고침 후에도 유지)
5. **코드 하이라이팅**: highlight.js 테마도 자동 전환 (github-dark ↔ github)
6. **색상 대비 개선**: `--text-dim` 색상을 WCAG AA 기준 충족하도록 밝게 조정
