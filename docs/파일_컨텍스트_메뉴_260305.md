# 파일 컨텍스트 메뉴 (삭제/이름변경) 개발 내역

> 개발일: 2026-03-05

---

## 개요

파일 트리에서 우클릭 컨텍스트 메뉴를 통해 파일 다운로드, 이름 변경, 삭제 기능을 사용할 수 있도록 구현했습니다.

## 변경 사항

| 파일 | 변경 내용 |
|------|----------|
| `web/routers/files.py` | `POST /api/rename`, `POST /api/delete` 엔드포인트 추가 |
| `web/static/index.html` | 컨텍스트 메뉴 HTML 추가 |
| `web/static/style.css` | `.context-menu` 스타일 추가 |
| `web/static/app.js` | `showContextMenu()`, `hideContextMenu()`, `contextMenuAction()` 함수 추가, 트리 아이템에 contextmenu 이벤트 바인딩 |

## API 명세

### POST /api/rename
- Request: `{ "path": "상대경로", "new_name": "새이름.md" }`
- 경로 순회 차단, 중복 파일명 409 에러

### POST /api/delete
- Request: `{ "path": "상대경로" }`
- 파일 및 폴더 삭제 지원
- 삭제 전 클라이언트에서 confirm 다이얼로그 표시

## 주요 기능

1. **우클릭 컨텍스트 메뉴**: 파일/폴더에서 우클릭 시 다운로드, 이름변경, 삭제 옵션 표시
2. **이름 변경**: 프롬프트 다이얼로그로 새 이름 입력
3. **삭제 확인**: 삭제 전 confirm 다이얼로그 필수
4. **보안**: 경로 순회(path traversal) 차단
