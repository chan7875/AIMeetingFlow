# 파일 검색/필터 기능 개발 내역

> 개발일: 2026-03-05

---

## 개요

사이드바 파일 트리에서 파일명으로 실시간 검색/필터링할 수 있는 기능을 추가했습니다.

## 변경 사항

| 파일 | 변경 내용 |
|------|----------|
| `web/static/index.html` | 사이드바 상단에 검색 입력 필드 및 초기화 버튼 추가 |
| `web/static/style.css` | `.search-box` 스타일, 검색 하이라이트 스타일 추가 |
| `web/static/app.js` | `filterTree()`, `clearSearch()` 함수 추가, `Ctrl+K`/`Cmd+K` 단축키 지원 |

## 주요 기능

1. **실시간 필터링**: 검색어 입력 시 파일명 기준으로 트리를 즉시 필터링
2. **폴더 자동 펼침**: 검색 결과에 매칭되는 파일이 있는 폴더를 자동으로 펼침
3. **단축키 지원**: `Ctrl+K` (Mac: `Cmd+K`)로 검색 입력에 포커스
4. **ESC 초기화**: 검색 입력에서 ESC 키로 검색어 초기화
5. **모바일 대응**: 단축키 사용 시 모바일에서도 사이드바 자동 열림
