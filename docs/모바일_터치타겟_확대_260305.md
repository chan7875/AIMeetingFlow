# 모바일 터치 타겟 확대 개발 내역

> 개발일: 2026-03-05

---

## 개요

모바일 환경에서의 터치 조작성을 개선하기 위해 버튼 및 인터랙티브 요소의 크기를 확대하고, GPU 가속 및 safe-area 대응을 적용했습니다.

## 변경 사항

| 파일 | 변경 내용 |
|------|----------|
| `web/static/style.css` | 모바일 미디어쿼리 내 터치 타겟 확대, GPU 가속, safe-area, AI 패널 높이 조정 |

## 세부 변경 내역

1. **헤더 버튼**: `padding: 8px 10px`, `min-width: 44px`, `min-height: 44px` (WCAG 터치 타겟 기준)
2. **트리 아이템**: `min-height: 40px`, 패딩 확대
3. **섹션 바**: `min-height: 44px`
4. **컨텍스트 메뉴**: `min-height: 44px`, 패딩 확대
5. **AI 패널**: 모바일 최소 높이 `40vh`로 조정
6. **사이드바 드로어**: `transform: translate3d()` + `will-change: transform`으로 GPU 가속
7. **safe-area**: `env(safe-area-inset-bottom)` 패딩 추가 (노치 대응)
