# 마크다운 TOC 자동 생성 개발 내역

> 개발일: 2026-03-05

---

## 개요

마크다운 파일 렌더링 시 H1~H3 헤딩을 기반으로 자동 목차(Table of Contents)를 생성하는 기능을 추가했습니다.

## 변경 사항

| 파일 | 변경 내용 |
|------|----------|
| `web/static/index.html` | 뷰어 상단에 TOC 컨테이너 (`#toc-container`, `#toc-list`) 추가 |
| `web/static/style.css` | TOC 스타일 (`.toc-item`, `.toc-h1`~`.toc-h3`), 헤딩 `scroll-margin-top` 추가 |
| `web/static/app.js` | `generateTOC()`, `toggleToc()` 함수 추가, 파일 열기 시 자동 TOC 생성 |

## 주요 기능

1. **자동 목차 생성**: H1~H3 헤딩에서 목차 항목 자동 추출
2. **앵커 스크롤**: 목차 항목 클릭 시 해당 헤딩으로 부드러운 스크롤 이동
3. **접기/펼치기**: 목차 헤더 클릭으로 목차 목록 접기/펼치기
4. **최소 헤딩 기준**: 헤딩이 2개 미만이면 TOC 미표시
5. **계층 표시**: H1/H2/H3 레벨에 따라 들여쓰기 적용
