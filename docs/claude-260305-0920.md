## 개선사항

### 🔴 즉시 — AI 결과가 원시 텍스트로 표시

**문제:** `result-area`에 AI 응답이 마크다운 없이 raw text로 표시됨. 헤딩, 코드블록, 목록이 모두 날 것의 기호로 보임.
**제안:** 스트리밍 완료(`done` 이벤트) 시 `marked.parse()`로 렌더링
**적용 위치:** `app.js` — `runAIStream()` done 핸들러
**난이도:** 낮음

---

### 🔴 즉시 — marked.js XSS 방어 없음

**문제:** `marked.parse(content)` 결과가 그대로 `innerHTML`에 삽입됨. vault에 악성 HTML/JS가 포함된 마크다운 파일이 있으면 실행됨.
**제안:** `marked` 옵션에 `sanitize: true` 설정 또는 `DOMPurify.sanitize()` 적용
**적용 위치:** `app.js` — `renderMarkdown()` 함수
**난이도:** 낮음

---

### 🟡 단기 — 이미지·PDF 파일이 사이드바 트리에 미표시

**문제:** `files.py:40` — 트리 빌드 시 `.md`, `.txt`, `.pptx`만 표시. 업로드한 이미지/PDF가 트리에 안 보임.
**제안:** 트리에 `.png`, `.jpg`, `.jpeg`, `.gif`, `.pdf`, `.svg`, `.webp` 추가. 뷰어에서는 이미지 `<img>` 태그로, PDF는 `<iframe>`으로 표시
**적용 위치:** `web/routers/files.py:40`, `app.js` — `openFile()`
**난이도:** 낮음

---

### 🟡 단기 — 프롬프트 템플릿 저장 기능 없음

**문제:** 커스텀 프롬프트를 매번 다시 입력해야 함. 기본 프롬프트가 HTML textarea와 Python에 이중 하드코딩됨.
**제안:** 사이드바 또는 별도 드롭다운으로 프롬프트 프리셋 저장/불러오기 기능 추가 (localStorage 활용)
**적용 위치:** `index.html` prompt-section, `app.js`
**난이도:** 중간

---

### 🟡 단기 — Auto-watch 오류가 UI에서 보이지 않음

**문제:** `_AUTO_WATCH_STATE["last_error"]`에 오류가 기록되지만, 헤더 `🛰 자동 분석` 버튼에는 아무런 표시 없음.
**제안:** `/api/ai/auto-watch` 폴링 시 `last_error`가 있으면 버튼을 주황색으로 바꾸고 툴팁에 오류 내용 표시
**적용 위치:** `app.js` — `updateAutoWatchBtn()`, `style.css`
**난이도:** 낮음

---

### 🟡 단기 — 설정 모달에 Issue 폴더 경로 없음

**문제:** `/api/config`가 `issue_folder`를 반환하지만, 설정 UI에서는 vault 경로만 변경 가능. Issue 폴더를 바꾸려면 서버 재시작 필요.
**제안:** 설정 모달에 Issue 폴더 입력 필드 추가
**적용 위치:** `index.html` settings-modal, `app.js` — `saveSettings()`
**난이도:** 낮음

---

### 🟡 단기 — Git push 시 git add -A가 민감 파일 포함 가능

**문제:** `git.py:103` — `git add -A`는 `.env`, credentials 파일 등 의도치 않은 파일을 스테이징할 수 있음.
**제안:** push 모달에 변경된 파일 목록을 미리 보여주고 체크박스로 선택 가능하게 변경
**적용 위치:** `web/routers/git.py`, `index.html` git-push-modal, `app.js`
**난이도:** 중간

---

### 🟢 장기 — 대용량 마크다운 파일 렌더링 성능

**문제:** 수천 줄의 마크다운도 전체 한 번에 `innerHTML`로 삽입. 큰 파일에서 UI 블로킹 가능.
**제안:** 파일 크기 임계치(예: 200KB) 초과 시 가상 스크롤 또는 청크 렌더링 적용
**난이도:** 높음

---

### 🟢 장기 — 업로드 폴더 피커가 평면 목록

**문제:** `upload-folder-picker`가 중첩 폴더를 제대로 탐색 불가.
**제안:** 트리 형식의 폴더 피커로 교체
**난이도:** 중간
