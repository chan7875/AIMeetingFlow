# 이미지/PDF 트리 표시 및 뷰어 미리보기 (2026-03-05)

## 변경 배경
- 기존 트리 API는 `.md`, `.txt`, `.pptx`만 노출하여 업로드한 이미지/PDF 파일이 사이드바에 보이지 않았습니다.
- 파일 확인을 위해 외부 앱을 열어야 해서 작업 흐름이 끊겼습니다.

## 적용 내용
- `web/routers/files.py`
  - `TREE_VISIBLE_EXTENSIONS`를 추가해 이미지/PDF 확장자를 트리 목록에 포함했습니다.
- `web/static/app.js`
  - 이미지/PDF 확장자 분기(`IMAGE_EXTENSIONS`, `PDF_EXTENSIONS`)를 추가했습니다.
  - 이미지는 뷰어 내 `<img>`로 렌더링, PDF는 `<iframe>` 미리보기로 렌더링합니다.
  - 텍스트 파일이 아닐 때 `AI 요약` 버튼은 숨기고 다운로드 버튼만 유지하도록 버튼 로직을 조정했습니다.
- `web/static/style.css`
  - 미리보기 전용 스타일(`.viewer-media-preview`, `.viewer-preview-image`, `.viewer-preview-pdf`)을 추가했습니다.

## 기대 효과
- 업로드한 이미지/PDF 파일을 트리에서 즉시 확인할 수 있습니다.
- 뷰어 화면 안에서 바로 시각 확인이 가능해 파일 검수 속도가 개선됩니다.
